# Hospital_management
The purpose of the project entitled as HOSPITAL MANAGEMENT SYSTEM is to computerize the Front Office Management of Hospital to develop software which is user friendly simple, fast, and cost-effective. It deals with the collection of patientâ€™s information, diagnosis details, etc. The Hospital Management System can be entered using a username and password. It is accessible either by an administrator or doctor. Only they can add data into the database. The data can be retrieved easily. The data are well protected for personal use and makes the data processing very fast. 

Extract the zip file and save it to the htdocs folder in the directory where the XAMPP is installed and then run the XAMPP server. 

Then open the file using the phpmyadmin after starting the server of apache. 

You will be able to see all the tables in the database and also you can run the file by running the xml file.
